In Flows, we have different classes involved like EntryFlowService , ExitFlowService ,TransFlowService.
EntryFlowService: In entry, Hit the cache and built scenario details object and execute hop level and global level services based on configuration and it route to next hop.
TransFlowService : In Trans, it will transform the data from source application to the target application format using WTX and it will call the corresponding interface WTX Map.
ExitFlowService : T he exit flow will trigger and Exit Flow will identify the destination application Queue and it will process the data into the Queu e.
